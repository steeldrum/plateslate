Welcome to PlateSlate!

PlateSlate is a web site that provides a free service to users in order to help them plan meals. In short users decide what plates are slated to be served over time.

The web site is currently undergoing beta tests and, as such, it is incomplete. Nonetheless some brave beta testers have already reported that it helps!

The full potential of the site is described by our host provider Collogistics as a project (or wiki as they call it). Building the site is a collaborative effort derived from mainly volunteers. The plans are to derive revenue from users interested in special features (not yet implemented) or folks who find the web app valuable enough to remit shareware fees. (We deeply appreciate your support!)
